The Musical Cipher

Text encoded in base-99 · expressed as a base-12 fraction · every digit is a musical note

plain text → cipher (base-99) → decimal (base-99) → fraction (base-12) → notes + graph
Each character and common word maps to a unique 2-symbol base-99 code. The full cipher string is treated as a base-99 decimal, converted to a reduced fraction, and expressed in base-12 — where every digit (0–9, A, B) maps directly to a musical note.
